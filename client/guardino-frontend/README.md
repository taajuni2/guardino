# Guardino Frontend
Das Guardino Frontend ist eine webbasierte Benutzeroberfläche zur Visualisierung und Verwaltung der vom Backend bereitgestellten Detection-Daten.

### Aufgaben des Frontends 
- Anzeige von Detection-Events
- Übersicht über registrierte Agents
- Darstellung von Agent-Status und Heartbeats
- Zentrale Sicht auf sicherheitsrelevante Ereignisse


### Verwendete Technologie
- Angular 
- TypeScript 
- HTML/SCSS
- REST-Kommunikation und Websocket



### Architektur 
- Single Page Application 
- Kommunikation ausschliesslich mit dem Backend
- Angular-Komponenten in Ordner getrennt (Services, Komponenten, Entities)


### Installation
Erfolgt über docker-compose.yml im Guardino Ordner.
